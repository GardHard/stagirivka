Обязательные продукты для здорового питания:
Овощи
Брокколи, тыква, цветная капуста, репка, огурцы. Готовьте блюда из тушеных смесей без добавления масла.
Мясо
Крольчатина, телятина, индейка, курица без кожи.Мясеые блюда ежедневно должны присутсвовать в питании в тушеном, запеченом или вареном виде.
Продукты из молока
Сыр, ряженка, простокваша, творог, йогурт без добавок, кефир. Желательно кушать их кормящей маме ежедневно либо через день.
